# Power BI Project 1

## 📊 Overview

This repository contains a **Power BI dashboard project** designed to
analyze and visualize data through interactive reports.\
The goal of this project is to transform raw data into meaningful
insights that support data‑driven decision making.

## 🚀 Features

-   Interactive Power BI dashboard
-   Data transformation using Power Query
-   Visual analytics with charts, KPIs, and filters
-   Clean and user‑friendly report layout
-   Insightful metrics for better analysis

## 🗂️ Project Structure

    PowerBI-Project/
    │
    ├── PowerBI-Project 1.pbix   # Main Power BI dashboard file
    ├── README.md                # Project documentation

## 🧰 Tools & Technologies

-   **Power BI Desktop**
-   **Power Query**

## ▶️ How to Use

1.  Download or clone this repository.
2.  Open the `.pbix` file using **Power BI Desktop**.
3.  Explore the dashboard and interact with filters and visuals.
